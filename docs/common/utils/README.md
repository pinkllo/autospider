# Utils (内部工具)

`common.utils` 模块包含一系列通用的底层工具函数，涉及延迟控制、文本搜索和路径管理。

---

## 📁 主要工具

### 1. `delay.py` (延迟控制)
提供带有随机抖动的延迟功能，用于模拟人类操作习惯。
- `human_delay()`: 基准延迟 + 随机抖动（由 `.env` 配置）。

### 2. `fuzzy_search.py` (模糊文字搜索)
在 HTML 或文本中进行不精确匹配。
- **用途**: 当精确 XPath 失效时，根据文字内容模糊定位元素。

### 3. `paths.py` (路径管理)
统一管理项目中的各种目录路径（如 `output/`, `logs/`, `screenshots/`）。
