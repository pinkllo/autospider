# 断点恢复与速率控制 (Checkpoint)

`checkpoint` 模块确保采集任务在中断后能够安全恢复，并根据目标网站的响应状态自动调整采集速率。

---

## 🚀 核心功能

### 1. `ResumeCoordinator` (恢复协调)
- **进度保存**: 实时将当前页码、已采集 URL 列表持久化到 Redis 或本地。
- **状态恢复**: 启动时自动检测 `Resume` 标志，跳过已完成的步骤。

### 2. `AdaptiveRateController` (自适应限速)
- **退避机制**: 当检测到反爬（如 403, 429）或加载超时时，自动增加操作延迟。
- **自动恢复**: 在连续成功一定次数后，尝试降级限速等级以提高效率。

### 3. `ResumeStrategy`
定义了不同的恢复场景和触发条件。
