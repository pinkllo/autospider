# Crawler Batch 模块

该模块负责执行基于配置文件的批量 URL 收集任务。

## 核心组件

### `BatchCollector` 
`BatchCollector` 继承自 `BaseCollector`，专门用于执行第二阶段的批量抓取任务。

#### 主要功能：
- **配置驱动**：从探索阶段生成的配置文件中加载抓取逻辑（如起始 URL、任务描述、导航步骤和 XPath）。
- **断点续爬**：支持从上次中断的位置恢复抓取。
- **自动化执行**：利用预定义的规则（XPath 等）高效地遍历列表页并收集详情页链接。

## 工作流程
1. **预加载配置**：读取 JSON 格式的采集配置。
2. **初始化环境**：设置基类所需的运行时数据。
3. **执行收集**：调用主流程开始批量抓取。
